Hope Future 博客介绍

 

欢迎使用 Hope Future 博客系统。请到这里来，请到这里来 ... 这里有一个实现梦想的世界。赶快开启您的博客吧！

世界是美好的！生活是美好的！在这个美好的时代，我们借助互联网，可以在这里展现自己的才华， 分享自己的快乐，发表自己的看法，书写自己喜欢的文章，研究自己喜欢的东西，追求自己的梦想，实现自己的价值。

最终，我们成为一个强者，来到这里，希望您能尽快成长。

该项目是一个免费blog搭建系统，项目完全是开源免费的，想学习研究的童鞋们请 猛击这里 来获取项目源码。项目开发过程中用到了时下比较流行的前端开源框架和技术。用到的相关技术主要有：

  • 项目管理：Grunt Bower Yeoman
  • 项目基于Bootstrap css框架开发
  • 后台服务器基于Node.js，利用 express 来搭建项目
  • 数据库采用 Mongodb，利用 mongoose 驱动来管理数据库
  • 页面渲染采用 jQuery、Angular、 Angular UI 等
  • 页面表单校验采用jQuery插件 jquery-validation
  • 单元测试：grunt + karma + Jasmine，基于Angular 测试
  • 端到端（e2e）测试：protractor
  • Javascript 书写和语法校验：JSHint
  • Javascript 文档（doc）生成： Jsdoc
