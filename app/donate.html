Hope Future 博客 - 爱心捐助

Hope Future Blog 系统既可以发表自己的文章，又可以拿来当作参考项目（ 源代码 ），开发自己的产品或项目。

开发本系统花费了我半年多的时间，从系统搭建、系统开发、系统测试再到系统上线，占用了我大部分业余时间。 如果您觉得本系统对您的学习有所帮助，请献上您的一份爱心，您的爱心会支撑我继续完成系统的升级开发和日常维护。 我也会把开发过程中用到的技术以博文的形式逐步呈现给大家的，同时会发布一些开源项目，方便大家的学习和交流。

该博客系统采用时下比较前沿的技术来开发，主要基于 Node.js 来实现的。用到的技术和资源有：

  1. Web 前端项目管理： Grunt、 Bower、 Yeoman
  2. JavaScript 框架和插件： jQuery、 Angular、 Angular UI Bootstrap、 jQuery Validation Plugin、 Moment.js、 Underscore.js、 TinyMCE、 SyntaxHighlighter
  3. Css 框架和工具： Bootstrap、 Font Awesome、 Less
  4. Web 开发辅助工具： JSHint、 JSDoc、 Jasmine 单元测试
  5. Node.js web application 框架或工具： express、 ejs、 crypto 加密、 nodemailer
  6. 数据库： MongoDB、 Mongoose
